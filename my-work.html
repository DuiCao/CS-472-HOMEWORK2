Project Overview: This app is designed to help students explore and join clubs at UW-Madison efficiently.

Purpose Description

  • Design Concepts Development
  • Prototype Creation
  • Personalized Club Survey
  • For You Page (FYP)

  2. Refine design ideas to enhance user experience.
  3. Develop interactive screens including Homepage, User Profile, and FYP.
  4. Collect user data for personalized club recommendations.
  5. Showcase recommended clubs based on interests.
